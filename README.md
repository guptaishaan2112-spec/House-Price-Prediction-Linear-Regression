# Linear Regression Project

## Overview
This project demonstrates a simple **Machine Learning model using Linear Regression**.  
The goal is to understand how a model learns relationships between input features and output values using Python and common data science libraries.

---

## What This Project Does
- Loads and explores a dataset
- Performs data preprocessing using Pandas and NumPy
- Splits data into training and testing sets
- Trains a Linear Regression model
- Makes predictions on unseen data
- Evaluates model performance

---

## Tech Stack
- Python 🐍
- Pandas
- NumPy
- Matplotlib
- Scikit-learn
